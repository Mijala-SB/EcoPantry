# EcoPantry
EcoPantry: Smart Inventory and Food Sharing application to Reduce Food Waste.

This is an Android application built with Kotlin and Android Studio to help Malaysian households reduce food waste through smart inventory management, community food donations, and AI-assisted meal planning.

This project is developed as part of BIT216 – Software Engineering Principles at HELP University (Semester May 2026).

Features of this project:
  1. Register & Privacy: Email registration, 2FA via OTP, privacy settings
  2. Food Inventory: Add, edit, delete, and categorise food items with expiry tracking
  3. Browse & Donate: View and claim community food donation listings
  4. Food Analytics: Visual dashboard of food saved and donations made
  5. Notifications: Expiry alerts, donation confirmations, meal reminders
  6. Meal Planner: Calendar-based weekly meal planning linked to inventory
